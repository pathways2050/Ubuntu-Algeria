Plugin to allow users leave comments on your blog using their facebook accounts

== Description ==

With that plugin you can replace default Wordpress comments to Facebook comments plugin, then, the visitor will be able to use their Facebook accounts to leave comments on your posts.

== Installation ==

Follow the below steps to intall the plugin:
1- Send plugins's folder to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin
3- Now, in the Settings area, a new link will appear "Facebook Comments", click on that link to go to plugin options page
4- Change the options you want and click on button Save Changes
5- Note that the facebook comments will appear on end of posts automatically after you activate the plugin, so, its a good idea deactivate default Wordpress comments on each post of your blog or edit the code of single.php manually to deactivate comments on all posts.
